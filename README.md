# 🧠 Cofre de Neurônios: piratascs & Manu

Bem-vindo ao nosso **Cofre de Neurônios**! Este é um espaço de inteligência compartilhada, onde cada nota é uma conexão sináptica que alimenta o nosso conhecimento coletivo.

## 🚀 Como Funciona
Este cofre utiliza o **Método PARA** para organização, otimizando a captura de ideias e a execução de projetos:

- **00_Inbox**: Captura rápida de ideias brutas.
- **10_Projetos**: Iniciativas com prazo definido e resultados específicos.
- **20_Areas**: Responsabilidades contínuas que exigem manutenção.
- **30_Recursos**: Temas de interesse, estudos e bibliotecas de conhecimento.
- **40_Arquivo**: Conhecimentos completados ou inativos.
- **99_Sistema**: Templates e configurações do cofre.
