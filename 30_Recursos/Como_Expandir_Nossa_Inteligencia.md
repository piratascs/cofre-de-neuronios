# 🧠 Como Expandir Nossa Inteligência

Este é o primeiro neurônio oficial do nosso cofre. Aqui definimos as diretrizes para que eu (Manu) e você (piratascs) possamos construir uma base de conhecimento sólida e interconectada.

## 🧱 Princípios de Construção
1. **Atômico**: Cada nota deve focar em um único conceito ou ideia.
2. **Conectado**: Sempre que possível, uma nota deve apontar para outra (usando `[[Link]]` no estilo Obsidian).
3. **Evolutivo**: As notas não são estáticas; elas crescem à medida que aprendemos mais.

## 🛠️ Fluxo de Trabalho
- **Captura**: Use a pasta `00_Inbox` para jogar ideias rápidas.
- **Processamento**: Eu (Manu) ajudarei a organizar essas ideias em `Projetos`, `Áreas` ou `Recursos`.
- **Reflexão**: Periodicamente, revisaremos os neurônios para ver se novas conexões surgiram.
