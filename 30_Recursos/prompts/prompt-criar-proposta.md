# Prompt para Criar Proposta Comercial

## Objetivo
Este prompt serve para gerar uma primeira versão de proposta comercial de marketing com mais contexto, clareza e lógica estratégica.

## Quando usar
Use este prompt quando precisar criar uma proposta para:
- gestão de marketing
- campanha
- consultoria
- projeto especial
- lançamento
- diagnóstico com plano de ação

## Como usar
1. Copie o prompt completo.
2. Cole na IA de sua preferência.
3. Substitua os campos entre colchetes.
4. Anexe ou cole materiais de contexto, se houver.
5. Revise a saída antes de enviar para qualquer cliente, sócio ou diretoria.

## Prompt base

Você é um estrategista de marketing sênior com foco em propostas comerciais claras, executáveis e persuasivas.

Crie uma proposta comercial em português do Brasil com tom profissional, objetivo e consultivo.

## Contexto
- Empresa/cliente: [nome]
- Segmento: [segmento]
- Situação atual: [descrever cenário]
- Problema principal: [descrever problema]
- Objetivo principal: [descrever objetivo]
- Público envolvido: [descrever público]
- Escopo desejado: [descrever escopo]
- Prazo esperado: [prazo]
- Verba disponível ou estimada: [verba]
- Restrições ou observações: [restrições]
- Diferenciais da solução: [diferenciais]

## Tarefa
Monte uma proposta comercial com a seguinte estrutura:
1. Contexto
2. Objetivo
3. Diagnóstico resumido
4. Solução proposta
5. Escopo de atuação
6. Entregáveis
7. Cronograma macro
8. Investimento ou premissas comerciais
9. Próximos passos

## Regras
- Não use linguagem genérica.
- Não faça promessa exagerada.
- Não escreva como agência genérica.
- Use linguagem executiva, simples e convincente.
- Deixe claro o que está sendo proposto.
- Conecte problema, solução e resultado esperado.
- Se faltar dado, sinalize o que precisa ser validado.
- Estruture o texto para facilitar adaptação em apresentação, PDF ou mensagem comercial.

## Formato de saída
Entregue em Markdown, com títulos e subtítulos bem organizados.
Ao final, inclua:
- uma versão resumida da proposta em até 8 linhas
- 3 possíveis títulos para essa proposta
- 3 riscos ou pontos de atenção
- 3 perguntas que eu deveria validar antes de enviar

## Qualidade esperada
A proposta deve parecer escrita por alguém que entendeu o negócio, organizou o problema e sugeriu uma solução viável, e não por uma IA genérica.

## Versão rápida
Se eu pedir uma versão curta, entregue:
- contexto
- objetivo
- solução
- escopo
- próximos passos

## Exemplo de comando final
Use as informações abaixo para criar uma proposta comercial:
[cole aqui o briefing, anotações, reunião, diagnóstico ou contexto]
